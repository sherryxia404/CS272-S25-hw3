CS272

CS272 is a course that introduces students new to Computer Science to Web Development! In this class, we use technologies such as WordPress, HTML, CSS, and JavaScript.

CS571

CS571 is a course that introduces students already familiar with Computer Science to web development, mobile development, and voice development! In this class, we use technologies such as React, React Native, and WitAI.

This class is open-source!
